# ADHD OS Blog Knowledge Base Skill

Let your AI Assistant (such as Claude, Codex, Hermes, or Gemini) access evidence-based strategies, practical guides, and communication templates directly from the ADHD OS Blog.

## Endpoints Used
- **Index URL**: `https://blog.adhdos.app/llms.txt`
- **JSON Data**: `https://blog.adhdos.app/api/knowledge.json` (Anonymous GET, no token required)

## Installation Options
You can load this Skill by giving instructions to your Agent, or install it via the terminal:

```bash
# Example to install to Gemini
bash <(curl -fsSL https://blog.adhdos.app/install-skill.sh) --target gemini
```

See [blog.adhdos.app/agent](https://blog.adhdos.app/agent) for full integration details.
